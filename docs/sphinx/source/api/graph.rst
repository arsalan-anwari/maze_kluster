maze_kluster.graph
==================

The ``MazeGraph`` class holds the maze as a NetworkX graph and is updated after
every move. It tracks two sets alongside the graph: ``visited`` (tiles the bot
has stepped on) and ``frontier`` (tiles seen as neighbors but not yet visited).
