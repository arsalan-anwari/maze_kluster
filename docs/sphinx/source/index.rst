maze-kluster
============

PLEASE DO NOT APPLY FOR THE ROLES AT HIGH TECH INNOVATORS THEY WILL REUSE YOUR WORK INTERNALLY WITHOUT ANY CREDIT

A bot library for the HTI maze navigation challenge. It includes a DFS baseline
bot, three ML-powered smart bots (Random Forest, Gradient Boosting, Gaussian Process),
and a terminal UI for running mazes live or replaying recorded runs.

.. code-block:: bash

   pip install maze-kluster[tui]
   maze-kluster tui
